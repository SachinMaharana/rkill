### Built With

- [Rust](https://www.rust-lang.org/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SachinMaharana/rkill.git
   ```
2. Build the project
   ```sh
   cargo build --release
   ```
3. Copy binary to bin folder
   ```JS
   cp ./target/release/rkill /usr/local/bin
   ```
4. Run
   ```
   rkill
   ```
